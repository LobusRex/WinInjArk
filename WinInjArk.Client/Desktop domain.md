# Desktop Domain
* MainWindow är skrivbordet
* Det finns knapp för utforskaren 
  * kan öppna hur många utforskare som helst
  * öppna utforskaren listar några filer
  * kan öppna en fil i texteditor
  * kan öppna samma fil i flera texteditorer
* det finns en knapp för epost
  * kan bara öppna epost-programmet en gång i taget
  * kan öppna epost i endast ett fönster/meddelande
  * kan skicka epost från huvudfönstret
